# Landing de reservas para productos de ciclismo

Abre `index.html` en el navegador para usar la landing. El formulario guarda las reservas en el navegador y permite exportarlas como `reservas-ciclismo.csv`.

## Plataforma recomendada

Para empezar rapido: Google Sheets.

Para algo mas fiable y profesional: Brevo o Mailchimp.

- Google Sheets es ideal para validar la idea y ordenar leads.
- Brevo es muy fuerte para email marketing, automatizaciones y listas con buen costo.
- Mailchimp es facil de usar y reconocido, pero puede subir de precio al crecer.

Mi recomendacion practica: usa Google Sheets como respaldo de leads y Brevo para enviar newsletters.

## Como guardar en Google Sheets

1. Exporta el CSV desde la landing.
2. Abre Google Sheets.
3. Crea una hoja nueva.
4. Importa `reservas-ciclismo.csv`.
5. Usa las columnas: Fecha, Nombre, Telefono, Gmail, Producto, Consentimiento.

## Siguiente paso para publicarla

Cuando tengas dominio o hosting, conecta el formulario con una de estas opciones:

- Google Forms o Google Apps Script para enviar directo a Sheets.
- Brevo Forms para capturar contactos y enviar newsletters.
- Mailchimp Forms si prefieres una herramienta mas conocida.

## Newsletter

Usa `newsletter-ciclismo.md` como plantilla diaria. Para una newsletter profesional, revisa fuentes oficiales y medios confiables cada manana, resume en tus palabras y agrega enlaces.
